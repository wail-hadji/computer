Random Access Memory

RAM It is known as reading and writing memory, and this is a type of temporary memory used in computers, as information is lost from it as soon as the power is cut off, so if the computer is restarted, for example, the information is lost.

Processor

It is a machine or electrical circuit that operates other electronic machines or circuits by giving them commands to perform operations or algorithms; Most of these operations are data processing. Processors are the main part of a computer, and they execute (software) commands.

RAM components

Each piece of memory is an integrated circuit composed of millions of cells formed by the union of transistors and capacitors, so that each transistor and capacitor constitutes one cell of the memory cells, and each cell of these cells is equivalent to one bit of data, and it is known that the bit is the smallest unit of measurement of memory and every 8 bits It constitutes one byte.

and a byte is the space sufficient to store the value of one letter, number, or symbol of ASCII characters.

Processor Basics

The CPU fetches instructions from memory, reads and writes instructions from memory, and transfers data to and from the input and output devices. The instruction cycle can be implemented in a simple and ideal way as follows:

  • Fetch the instruction to be executed from the memory whose address is stored in the PC register and stored in the IR register.
  • Decode the instruction.
  • Fetch variables from memory and store them in CPU registers.
  • Execute the instruction.
  • Transfer results from CPU registers to memory.
